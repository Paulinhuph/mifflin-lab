# ​⚖️ Mifflin - Lab
​O Mifflin - Lab é uma aplicação web de alta precisão projetada para o cálculo de parâmetros biométricos essenciais: a Taxa Metabólica Basal (TMB) e o Gasto Energético Total (GET).

## ​🧪 Base Científica
​A lógica de cálculo deste sistema utiliza a Equação de Mifflin-St Jeor, validada em 1990 e considerada o padrão-ouro para estimativas metabólicas em indivíduos saudáveis na era moderna.
- TMB (Homens) = (10 \times peso) + (6.25 \times altura) - (5 \times idade) + 5

- TMB (Mulheres) = (10 \times peso) + (6.25 \times altura) - (5 \times idade) - 161

## 🛠️ Tecnologias Utilizadas

- ​HTML5: Estruturação semântica de dados.

- ​CSS3: Estilização avançada com Grid Layout, Flexbox e Media Queries para total responsividade.

- ​JavaScript (Vanilla): Manipulação dinâmica do DOM, gerenciamento de eventos e lógica de cálculo matemático.

- ​FontAwesome: Iconografia técnica.

- ​Google Fonts: Tipografia refinada (Inter).

## ​🚀 Funcionalidades
- ​Cálculo de TMB: Estimativa de calorias queimadas em repouso absoluto.

- ​Cálculo de GET: Integração do fator de atividade física para determinar a necessidade calórica diária real.

- ​Design Responsivo: Interface adaptável para Desktop, Tablets e Smartphones.
​Seletor de Gênero Dinâmico: Experiência otimizada com botões de alternância de estado.
